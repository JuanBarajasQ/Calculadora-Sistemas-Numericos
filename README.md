# Calculadora Sistemas Numéricos en Java
#### Realizado por Juan Carlos Barajas Quintero 1152455 y Bryan Gelvez 1152500. Arquitectura de Computadores 1155504-C. 
<div align="justify">
Calculadora de 2 valores que permite operar en Binario, Octal, Decimal y Hexadecimal por medio de la consola. Se debe ejecutar el archivo Vista.java. Se indica la base a trabajar (2 -> Binario, 8 --> Octal, 10 --> Decimal, 16 --> Hexadecimal), el primer número y el segundo número (ambos en el sistema numérico seleccionado), y la operación a realizar (+ --> Suma, - -> Resta, * --> Multiplicación, / --> División).
